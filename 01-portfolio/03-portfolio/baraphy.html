design

main page

메인페이지에서는 신제품을 가장 상단에 배치하여 눈에 잘 띄도록 하였고 제품 카테고리와 베스트셀러, 인스타그램 이미지를 노출시킴으로서 다양한 정보를 제공한다.
네비게이션의 형태가 디바이스의 환경에 맞추어 사용하기 편리하도록 디자인하였다.

publishing

coding

메인에 이벤트배너를 setinterval을 이용하여 자동으로 슬라이드 되는 형식을 사용하였고 아래의 newpoduct는 화살표를 클릭하면 옆으로 넘어갈수 있도록 작업하였다.

반응형 웹 구현을 위해서 퍼센트를 이용하여 가로값을 주었고, 네비게이션의 경우 각 디바이스의 환경에 맞추어 웹에서만 2depth로 보이며 모바일에서는 햄버거메뉴를 이용하여 메뉴가 나타날수 있도록 하였다. 또 모바일의 경우 웹이나 태블릿과는 다르게 세로가 긴 환경을 고려하여 정렬의 배치를 적절히 바꾸어 구현하였다.

develop

태블릿이나 모바일 버전에서는 불필요해지는 hover기능을 mouseenter와 addclass를 이용하여 구현할수 있는 방법을 페이지를 모두 제작하고 난 후에 발견하여 다음 반응형 웹 구현 시 적용시켜 볼 생각이다.
